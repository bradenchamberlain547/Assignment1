UX and UI

This website was made to show what UX and UI. It also differentiates the designers jobs of UI and UX.

UI

UI is short for User Interface. The user interface is how a user navigates and uses a device. This may be by using apps, buttons, toggles etc. It is also what the device looks like. The UI designers jobs focus on:

  • Color Theory
  • Design Patterns
  • Interactivity/Animation

UX

UX is short for User Experience. Without a user interface, it would be very difficult to have user experience. User experience is the entire process a user is interacting with a product. A UX designers job focuses on:

  • User Research
  • Information Architecture
  • Testing the Program
